# Ako stiahnúť Flutter
- treba nainštalovať cez ofiko stránku podľa návodu: https://docs.flutter.dev/get-started/install
- po inštalácii overiť v terminály: flutter doctor

# Stiahnúť projekt
- git clone https://github.com/karolina3jurackova/MindStep.git
- cd mindstep

# Inštalácia knižníc
- flutter pub get

# Ako spustiť projekt vo VS code?
- Ísť do View -> Flutter: Launch Emulator -> napr Pixel 6
- Spustiť run
